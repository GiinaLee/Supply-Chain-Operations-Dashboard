# Retail Supply Chain & Operations Dashboard

## Project Objective
This project analyzes retail supply chain data to identify logistics bottlenecks, optimize continuous improvement strategies, and evaluate regional profitability.

## Tech Stack & Workflow
* Data Processing: Python (Pandas, NumPy) for feature engineering (Lead Time calculation, Returned Rate, Profit Rate generation).
* Data Visualization: Google Looker Studio for interactive operational dashboards.
* https://datastudio.google.com/reporting/6757b24f-b410-42bf-9c1e-d047fe86fab7

## Key Business Insights

### 1. Lead Time Distribution
* Standard items like 'Paper' and 'Binders' unexpectedly show a high rate of long lead times (>7 days).

### 2. Quality Control & Returns
* Binders and paper are the top contributors to our total return volume.

### 3. Regional Profitability vs. Logistics
* The West and East regions are the main drivers of our profit.
